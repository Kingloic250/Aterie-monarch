# Assets Directory

Place real media files here:

- `hero-poster.jpg` — poster image for the hero background video
- `hero-video.mp4` — looping background video for the hero section
- `og-image.jpg` — Open Graph social share image (1200x630 recommended)
- Project thumbnails, gallery images, team photos, client logos

All current sections use CSS gradients as placeholders. Replace the inline
gradient styles with `<img>` tags pointing to files in this directory.

Example replacement for a bento card thumbnail:
```html
<img src="assets/project-rwanda-rising.jpg" alt="Rwanda Rising documentary still" loading="lazy" />
```
